# RENADS - Documentación de Alcance para Implementación

## Proyecto

**Registro Nacional de Articulación Docencia-Servicio en Salud (RENADS)**

## Propósito de la documentación

Este conjunto de documentos define el contexto funcional, alcance, actores, módulos, reglas de negocio, casos de uso y requerimientos iniciales del proyecto RENADS. Servirá como base para la posterior fase de análisis detallado, diseño técnico, arquitectura de software e implementación.
